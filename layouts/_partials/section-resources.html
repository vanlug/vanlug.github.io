Design system

VanLUG's design system is based on PatternFly{{ partial "icon" "external" }}, the open-source design system used for this website.

Icon & logo

The VanLUG icon is a modified version of Tux Flat SVG{{ partial "icon" "external" }} by gg3po and Iwan Gabovitch, based on NewTux.svg{{ partial "icon" "external" }} by gg3po, itself based on the original Tux.png{{ partial "icon" "external" }} by Larry Ewing. The wavy stripes from the flag of Vancouver{{ partial "icon" "external" }} have been added to the belly. Licensed under the GNU GPL v2{{ partial "icon" "external" }} or later. The full logo combines the icon with the “VanLUG/Vancouver Linux User Group” wordmark, as seen on the banner and event cards.
